💡 사용 방법

1

젠데스크 티켓 페이지 열기

2

북마크 클릭 → 케이스 자동 감지 + 문의 내용 자동 추출

3

이용자/파트너 선택 → 🔍 매크로 추천으로 관련 매크로 확인 (선택사항)

4

복사하기 → 슬랙에 붙여넣기

🔧 최신 수정 내역

  • 🔴 수정 메시징 티켓 파싱: 마지막 드라이버 상담사 이후 Web User 메시지만 정확히 추출
  • 🔴 수정 케이스 감지 순서: 통행료 키워드 요금정정보다 우선 처리
  • 🔴 수정 복사 양식에서 케이스명 제거 → #번호 이용자/파트너 인입 형태
  • 🟡 교체 AI 분석 버튼 → 🔍 매크로 추천 (키워드 기반 즉시 추천, 외부 API 불필요)
  • 🟡 개선 이용자/파트너 버튼 UI 단순화
  • 🟡 추가 ESC 키로 팝업 닫기
