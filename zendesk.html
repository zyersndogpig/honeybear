💬 추천 멘트 — 이렇게 씁니다

상담사가 매크로로 답변을 작성하면, 도구가 고객 인입 문의와 작성 중인 답변을 함께 읽어 어울리는 보조 멘트(중개 플랫폼 한계, 능동 제재 등)를 추천합니다. 답변을 아직 안 썼어도 문의 내용만으로 관련 멘트가 위로 정렬됩니다.
  • 추가 파트너 도움말 안내 멘트 — 드라이버 도움말/공지 참고 유도. 끝줄 '도움말 - [도움말 링크]'는 상담사가 어드민 링크로 채우는 자리표시(토글 아님)
  • 추가 콜 수요 징징이 1차 / 2차 멘트 — 콜 수요 부족 호소에 대한 공감 + 운영팀 전달 응대. 재문의(2차)는 문구를 달리해 반복 답변 티가 나지 않도록 분리
  • 추가 서울시 임산부 교통비 바우처 멘트 — 타다는 '택시' 업종 분류로 혜택 이용 가능, 단 토스페이 등 간편결제 결제 시 미적용 가능 안내 + 카드 직접 등록 경로 포함
  • 개선 배차 관련 불만(파트너) 멘트 문구를 공감 도입부 + 대시보드 콜 수요 지역 안내 + 운영부서 전달로 개편했습니다. 날씨 인사말 체크박스(☀️여름·❄️겨울·🌧️레인)를 추가해 체크 시 마무리 인사가 답변 맨 아래에 붙습니다(독립 토글, 해제 시 해당 문구만 제거)
  • 추가 배차 관련 불만(파트너)(진술햄 부류 전용) 멘트 — 공감 도입부 없이 배차 추천 구조 + 가까운 위치라도 다른 파트너 우선 추천 가능 안내에 집중
  • 개선 추천 점수에 가중치를 적용했습니다. 결정적 단서(우회·영업손실비·가결제 등)는 크게, 보조어는 작게 반영하고 빈출어(불편·확인 등)는 점수에서 제외해 ★ 정확도를 높였습니다
  • 개선 칩을 누르면 아래 칸에 누적됩니다(덮어쓰지 않음). 사과 시작 + 본문 + 마무리처럼 여러 멘트를 쌓아 한 번에 복사하세요. 직접 수정한 내용도 보존됩니다
  • 추가 상단 검색창으로 멘트를 라벨로 바로 찾을 수 있고, 비우기로 누적 칸을 초기화합니다
  • 개선 📋 슬랙 적재(티켓 복사)를 패널 상단으로 올렸습니다. 추천 멘트가 길어져도 티켓 단순 복사 시 스크롤 없이 바로 쓸 수 있습니다
  • 추가 공항 예약시간 변경 멘트 추가 — 예상요금 확정·드라이버 직접 수락 구조 + 입국/수하물 여유 안내 + ±30분·10분 단위 변경 경로/유의사항 + 도움말 링크
  • 추가 카시트 환불관련 멘트 — 드롭다운 경우에 카시트 미설치 환불 추가(이너시트 미제공·카시트 미설치·카시트 미숙지·카시트 오염·환불 불가)
  • 개선 카시트 환불 멘트의 운행요금 전액 환불 불가 문단을 대괄호 토글로 전환했습니다. 기본은 빠진 상태로 나오고, 필요할 때만 상단 체크박스로 추가합니다
  • 개선 멘트 24종으로 정리 — 파트너 대응 멘트(셔틀 불만·만족도 불만족·배차 추천 구조 안내)와 공항 예약시간 변경·카시트 환불 추가, 사용 빈도 낮은 멘트(취소/노쇼 수수료·엄마포인트 후적용·영손비·분실물) 정리
  • 개선 선택 문구 포함 체크박스는 문장형 대괄호가 있는 멘트를 누를 때만 표시되며, 라벨이 멘트에 맞게 자동 변경됩니다 — 드라이버 빠따에서는 "영구 차단 포함", 카시트 환불에서는 "운행요금 전액 환불 불가 포함"만 표시됩니다. [ ] 같은 짧은 채워넣기 표시는 항상 유지됩니다
  • 개선 카시트 환불 멘트의 연결어를 다듬어 운행요금 전액 환불 불가 문단을 넣든 빼든 자연스럽게 이어지도록 정리했습니다(그럼에도 → 아울러, 미설치 중복 시작어 보정)
  • 패널의 🔄 다시 읽기로 현재 답변 에디터 내용을 다시 읽어 추천을 갱신합니다
