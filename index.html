💡 사용 방법

1

타다 어드민에서 라이드 또는 예약 페이지 열기

2

북마크 클릭 → 탭 선택 (분실물 / 통행료 / 요금 정정 / 영손비 / 수수료환불 / 수수료청구)

3

내용 확인 → 복사하기 → 문자 발송 툴에 붙여넣기

4

예약 파생 건 요금정정/영손비/수수료 탭: 예약 → 라이드 순으로 2단계 실행
단, 수수료환불은 파생 라이드 없으면 예약에서 바로 복사 가능

🔧 최신 수정 내역

  • 🟡 추가 요금 정정 문자에 "할인 및 크레딧 미적용 기준" 안내 문구 추가
  • 🔴 수정 단독 실행 파싱을 꿀통 양식과 통일 (영수증 +항목 합산 우선, 할인·크레딧·계좌이체·포인트 제외) + 기존/정정 금액 현재 파싱값 우선
  • 🔴 수정 수수료환불: 파생 라이드 없는 예약건 → 실제요금 직접 파싱 후 바로 복사
  • 🔴 수정 수수료환불: 라이드 없을 때 버튼 "복사하기"로 정상 표시
  • 🔴 수정 통행료: 의왕통행료 1,000원 추가
  • 🟡 추가 영손비 탭: 이용자 문자 코드블록 함께 복사 체크박스
  • 🟡 추가 호출예약 파생 라이드에서 요금정정/영손비/수수료 탭 선택 시 예약 먼저 실행 안내
  • 🟡 수정 예약 파생 여부 감지 개선 (rideId 빈값/대시 처리)
