💡 사용 방법

1

라이드/예약 페이지에서 북마크 클릭 → 탭 타이틀 📍 라이드 저장됨 확인

2

유저 페이지에서 북마크 클릭 → 👤 유저 저장됨 확인

3

파트너 페이지에서 북마크 클릭 → 팝업 자동 출력

4

요금 정정 필요 시 라이드에서 한번 더 클릭 → 요금 자동 파싱 후 팝업 재출력

5

템플릿 선택 → 내용 수정 → 복사하기 → 슬랙/넥스트 붙여넣기

🔧 최신 수정 내역

  • 🔴 수정 라이드 추가 실행 시 유저/파트너 ID 유지 (요금 정보만 업데이트)
  • 🔴 수정 라이드 추가 실행 시 예약 운행 정보 ID 불일치 → 휴먼에러 팝업 + 초기화 버튼
  • 🔴 수정 예약 단독 실행 시 다른 건 ID 잔존 방지 (isFollowUp 조건 강화)
  • 🟡 수정 결제요금 미감지 시 노란 배경 경고 + 상태바 회색 표시
  • 🟡 수정 라이드 미수집 시 요금 정정 템플릿에 경고 표시
  • 🟡 수정 실제요금 파싱 방식 개선 (총N원 → =N → 단순숫자 순)
  • 🟡 추가 플러스/넥스트 자동 감지 → 템플릿 자동 변환
  • 🟡 추가 토스/티머니/비회원 현장결제 자동 태그
  • 🟡 추가 예약 파생 라이드 경고 팝업에 ID 초기화 버튼
