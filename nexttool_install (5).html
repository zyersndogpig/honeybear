💡 사용 방법

1

라이드 또는 예약 페이지에서 북마크 클릭 → 📍 라이드 저장됨 탭 타이틀 확인

2

유저 페이지에서 북마크 클릭 → 👤 유저 저장됨 확인

3

드라이버 페이지에서 북마크 클릭 → 팝업 자동 출력 (드라이버 없으면 유저까지만 해도 팝업 뜸)

4

템플릿 선택 → 내용 수정 → 복사하기

🔧 수정 내역

  • 🔴 수정 드라이버 미배정 시 팝업 영구 미출력 → 유저+라이드만 있어도 팝업 표시
  • 🔴 수정 팝업 없을 때 수집 상태 표시 (뭐가 부족한지 안내)
  • 🔴 수정 extra 입력창 토글 버튼 추가 (항목별 내용 추가/숨기기 가능)
  • 🔴 수정 blinkTitle 함수 선언 순서 수정 (복사 완료 시 정상 동작)
  • 🟡 수정 이전 건 ID 잔존 문제 → 복사 완료 시 자동 삭제 / 취소 시 유지
  • 🟡 추가 ID 초기화 버튼 추가
  • 🟡 수정 execCommand listener once:true + Clipboard API fallback
  • 🟡 수정 알림 순서 개선 — 복사 완료 확인 후 넥스트/플러스 안내
  • 🟡 수정 URL 패턴 소문자 ID 매칭 추가
  • 🟡 추가 선택된 템플릿 행 하이라이트
