꿀통 × 꿀빠는 곰 연동 패치

넥스트 적재 양식과 고객 문자 발송 도구를 하나로

2026.05.29 업데이트

한 번 긁으면, 두 번 안 긁는다

이제 꿀통에서 라이드·예약을 처리하면 고객 정보가 자동 저장되어, 꿀빠는 곰에서 라이드든 예약이든 어디서 실행해도 바로 문자 양식이 떠요. 꿀통을 안 쓴 경우(남이 적재한 슬랙에서 문자만 보낼 때)엔 기존처럼 예약↔라이드를 오가며 수집해요.

🍯 꿀통 적재 → 정보 자동 저장 → 🐻 꿀빠는 곰 즉시 실행

📌 북마크 설치

아래 버튼을 북마크바로 드래그하면 설치 완료!

🍯 꿀통 양식 🐻 꿀빠는 곰 🎫 티켓 뷰
⭐ 핵심 신규 기능
1 꿀통 ↔ 꿀빠는 곰 데이터 연동NEW
꿀통이 라이드·예약을 긁을 때 고객명·탑승시각·출발/도착지·요금 등을 자동 저장합니다. 꿀빠는 곰 실행 시 현재 페이지가 저장된 건과 일치하면 2단계 과정 없이 바로 문자 양식을 띄워요.
  • 요금 정정 금액(기존/정정)도 꿀통에서 그대로 넘어옴
  • 항목별 정정 내역(카시트·거리추가 등)까지 자동 입력
  • 분실물 물품명도 꿀통 입력 시 꿀빠는 곰에 자동 입력
  • 꿀통에서 선택한 탭이 꿀빠는 곰에서 자동 선택됨
2 휴먼에러 방지 경고창NEW
꿀통으로 저장한 건과 다른 라이드·예약에서 꿀빠는 곰을 실행하면 경고창이 떠요.
  • 저장된 정보로 진행 / 현재 페이지로 진행(저장값 리셋) / 취소 중 선택
  • 실수로 엉뚱한 건의 문자를 보내는 사고 방지
3 통행료 변경 기능NEW
통행료 청구뿐 아니라 금액 변경(예: 2,000원 → 3,500원)도 지원해요.
  • 기존 통행료는 영수증에서 자동 파싱
  • 구간 선택 시 변경 통행료 자동 합산 계산
  • 기타 직접입력 여러 개 추가 가능 (목록에 없는 톨게이트 여러 곳 한 번에)
🔧 주요 개선 & 버그 수정
4 예약 파생 라이드 ID 처리FIX
예약에서 파생된 라이드는 호출 예약 ID를 메인으로 잡아 하이퍼링크를 걸어요. 파생 없는 순수 라이드는 라이드 ID를 메인으로 사용합니다.
5 예약 단독 실행 시 요금 처리FIX
라이드 미수집 예약 단독 실행 시, 예상요금을 결제요금으로 잘못 잡던 문제를 수정했어요.
  • 일반 요금은 비워두고, 취소수수료는 정상 표기(예약에 실제로 찍히는 값)
6 다른 라이드 데이터 누출 차단FIX
이전 라이드의 요금·통행료·탭 정보가 다른 라이드에 남던 문제를 수정했어요.
  • 통행료 기존금액은 현재 페이지 영수증에서 직접 파싱
  • 요금정정·항목·탭 자동선택 모두 라이드/예약 ID 일치 검증
7 크레딧 전액 차감 건 요금 인식FIX
엄빠택 크레딧 등으로 총액이 0원인 라이드에서, 할인 전 실제 청구 금액을 영수증 항목 합산으로 정확히 인식하도록 개선했어요.
8 주소·줄바꿈 파싱 개선IMP
  • 분실물 양식을 습득 공유 / 확인 요청 두 건으로 분리
  • 영문 주소의 중첩 괄호 깨짐 수정 (예: 1(il)-ga)
  • 복사 시 항목별 내역 줄바꿈 적용 (슬랙·넥스트 호환)
  • 항목명 가독성: "거리추가요금" → "거리추가 요금"
9 티켓 뷰 — 다중 인입 선택NEW
고객이 한 티켓에 여러 번 문의를 남긴 경우, 인입별 선택 버튼이 떠요.
  • 원하는 인입을 복수 선택 → 인입 순서대로 자동 합쳐짐
  • 메시징 티켓은 마지막 상담사 답변 이후 고객 발화만 수집
  • 이메일 티켓은 인용된 원본 메일 제외하고 고객 문의만 추출
10 티켓 뷰 — 다중 탭 격리FIX
여러 티켓 탭을 동시에 열어둬도 현재 보고 있는 티켓만 정확히 파싱해요.
  • 활성 탭의 대화 영역만 선택 (다른 탭 내용 섞임 방지)
  • 다른 탭에서 실행 시 안내 후 중단
11 정정 사유 "기본" 옵션 추가IMP
요금 정정 문자의 사유 선택에 기본(사유 없음)을 추가하고 기본값으로 설정했어요. 문장이 자연스럽게 이어지도록 처리됩니다.
